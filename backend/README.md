# RESOURCES PEN - Enterprise Backend Architecture

Modular Monolith built for high-performance digital asset marketplace orchestration.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express
- **Intelligence**: Gemini API
- **Security**: JWT + Helmet + Rate Limiting
- **Deployment**: Docker

## Getting Started
1. `cd backend`
2. `npm install`
3. `npm start`

## API Nodes
- `/api/v1/auth`: Identity Protocols
- `/api/v1/products`: Asset Manifests
- `/api/v1/orders`: Fiscal Settlement
- `/api/v1/support`: Helpdesk Triage
